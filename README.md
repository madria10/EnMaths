    <h1>Engineering Mathematics Practicals - SEM 3 | 2024 - Delhi University</h1>
    <p>
        This repository contains practical programs and scripts developed for the 
        <strong>Engineering Mathematics</strong> course using Scilab. The focus is on applying mathematical concepts 
        to solve equations, model systems, and analyze results effectively.
    </p>
    <h2>Learning Outcomes</h2>
    <p>By completing this set of practicals, learners will be able to:</p>
    <ul>
        <li>Perform operations with various forms of complex numbers to solve equations.</li>
        <li>Use mathematics as a tool for solving and modeling systems in electronics.</li>
        <li>Prepare comprehensive technical reports on experiments and results.</li>
    </ul>
    <h2>Practical Component Overview</h2>
    <p>This course involves <strong>30 hours</strong> of hands-on practice with mathematical tools and software.</p>
    <h3>List of Practicals</h3>
    <ol>
        <li><strong>Solution of First-Order Differential Equations:</strong> Learn to solve differential equations using appropriate numerical methods.</li>
        <li><strong>Testing Convergence of a Series:</strong> Write and implement algorithms to test the convergence of given series.</li>
        <li><strong>Testing Divergence of a Series:</strong> Implement methods to check if a given series diverges.</li>
        <li><strong>Solving Linear Systems of Equations (Gauss Elimination Method):</strong> Use the Gauss Elimination technique to solve linear systems efficiently.</li>
        <li><strong>Solving Linear Systems of Equations (Gauss-Seidel Method):</strong> Apply iterative Gauss-Seidel methods for solving linear systems.</li>
        <li><strong>Solving Linear Systems of Equations (L-U Decomposition Method):</strong> Utilize L-U decomposition to find solutions to linear equations.</li>
        <li><strong>Visualization of Functions and Taylor Series Expansions:</strong>
            <ul>
                <li>Plot exponential, logarithmic, and trigonometric functions.</li>
                <li>Compare these plots with their Taylor series expansion up to the first 10 terms.</li>
            </ul>
        </li>
    </ol>
